# 用語集

このドキュメントは、ゲーム全体で使用されるUIテキストの一貫性を保つための用語集です。

## 対象読者

- 開発者
- UI/UXデザイナー

## 基本方針

- 漢字を避け、ひらがなやカタカナを主に使用します。
- 幼児でも直感的に理解できる、シンプルで分かりやすい言葉を選びます。

## ゲームの状態

| 項目 | 日本語 | 備考 |
| :--- | :--- | :--- |
| ゲーム中 | ～のばん | 例：「あなたのばん」 |
| 勝利 | ～のかち！ | 例：「あなたの勝ち！」 |
| 敗北 | ～のまけ | 例：「あなたのまけ」 |
| 引き分け | ひきわけ | |
